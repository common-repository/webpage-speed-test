== Description ==

This Plugin is used to check webpage performance using the Google Insight API. This plugin also allows us to Increase webpage performance by using a script like Enable GZIP Compression, Leverage Browser Caching and defer javascript. Use given script In your htaccess and function.php file to boost your webpage speed.

== Installation ==

1. Upload `webpage-speed-test` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter Website URL and select strategy like desktop/mobile get all website performance report.
4. Use performance increase script to increase performance.
